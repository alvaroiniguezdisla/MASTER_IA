# Reto proyecto: Practica los foundation models en NLP

Proyecto en notebook para generar descripciones SEO estructuradas con OpenAI Structured Outputs y validarlas con Pydantic.

## Archivos

- `reto_proyecto_foundational_models_nlp.ipynb`: notebook principal para entregar.
- `requirements.txt`: dependencias necesarias.
- `sample_outputs/`: carpeta reservada para ejemplos de salida.

## Uso

1. Instala dependencias:

```bash
pip install -r requirements.txt
```

2. Abre el notebook en Jupyter o Google Colab.
3. Ejecuta todas las celdas. Por defecto funciona en modo demo sin API key.
4. Para usar API real, configura `OPENAI_API_KEY` y cambia `USE_REAL_API = True`.

## Entrega

El enunciado indica subir únicamente el archivo `.ipynb` al repositorio de GitHub.
